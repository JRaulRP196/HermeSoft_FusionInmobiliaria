# HermeSoft_FusionInmobiliaria
Este es un proyecto realizado para la empresa fusión inmobiliaria el cual busca mejorar y optimizar el procesos de ventas ofreciendo calculos, información, reportes y estadísticas en cuestion de segundos. Este proyecto optimizara los procesos de venta en cuanto al factor tiempo para aumentar así la cantidad de atención a clientes y ventas.
